Star Wars JavaScript Game

Description:
This game is a simple Star Wars-themed game built using JavaScript. Players control a character and engage with enemies by attacking using the arrow keys.

Gameplay Instructions:
Press the up arrow key to attack the enemy.
Press the > (right arrow) key to slide towards the right.
Press the < (left arrow) key to slide towards the left.
Refresh the webpage to restart the game.
Your score will be displayed as you shoot down enemies.

How to Play:
Download/Clone Repository
Open the index.html file in your preferred web browser.
Use the specified keys to control your character.
-----OR------
Just click on following link: https://star-wars-game-phi.vercel.app/

Enjoy the game and try to achieve the highest score!

Technologies Used:
JavaScript

Folder Structure:
index.html: Main HTML file for the game.
scripts/: JavaScript files for game logic.
styles/: CSS files for styling the game.
